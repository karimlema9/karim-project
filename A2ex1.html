Taula de multiplica

        import java.util.Scanner;
 
        public class tablaMultiplicar
        {
            public static void main(String[] ARGS)
            {
                Scanner obtenerNumero = new     Scanner(System.in);
                int numero,i,j;
 
                System.out.print("indica la taula de multiplicar: ");
                numero = obtenerNumero.nextInt();
 
                for(i = 1; i < =numero; i++)
                {
                    for(j = 1; j < = 12; j++)
                    {
                        System.out.println(i + " X " + j + " = " + i*j);
                }
                    System.out.println();
                }
            }
        }

Java és un llenguatge totalment orientat a objectes, això representa que posseeix instruccions i sintaxi específiques per a la programació orientada a objectes.

A més en java existeix el concepte de jerarquia d'herència, és a dir, que totes les classes han d'heretar d'una altra classe per formar totes elles un arbre invertit.

La classe arrel d'aquest arbre és la classe java.lang.Object i totes les classes hereten d'ella directament o indirectament.

A més a les classes són col·locades en carpetes per facilitar la seva ordenació i el treball amb elles, les carpetes (paquets és el terme més exacte) formaran part del nom de la pròpia classe, per exemple, java.lang.Object vol dir que hi ha una carpeta anomenada lang, dins de la qual hi ha una altra carpeta anomenada lang, dins de la qual hi ha una classe anomenada Object
